Toggle is a input component for flipswitch. You can use it directly like this:

```jsx static
import { Toggle } from 'inplayer-ui';
import { Label } from 'inplayer-ui';

function Page(props) {
  return <Toggle id="toggle1" />
<Label htmlFor="toggle1">Checkbox1</Label>
}
```

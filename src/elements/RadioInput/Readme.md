RadioInput is a input component for radio. You can use it directly like this:

```jsx static
import { RadioInput } from 'inplayer-ui';
import { Label } from 'inplayer-ui';

function Page(props) {
  return <RadioInput id="radio1" />
  <Label htmlFor="radio1">Radio1</Label>
}
```

### Demo

```jsx

  <Label htmlFor="radio1">Radio1</Label>
```
